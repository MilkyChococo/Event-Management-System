About Us

EventHub Verify keeps event discovery, registration, and admin control in one workflow.

The project was built for SE113.Q21 to make event information easier to trust, easier to update, and easier to manage from both attendee and admin views.

Focus

Cleaner event flow

Browse events, inspect details, reserve seats, and keep tickets in one place.

Reliability

MongoDB-backed updates

Admins can add, edit, and remove event data without hard-coding page content.

What We Help With

For attendees

  • Find events faster with filters, spotlight slides, and clearer ticket information.
  • Review venue, timing, organizer, and event details before reserving a seat.
  • Track reservations, ticket status, and account information from one account page.

Why It Feels Trustworthy

For the team behind the board

  • Event details are stored in MongoDB, so edits stay consistent across the site.
  • Admins get a dedicated manager flow for create, edit, delete, and analytics.
  • Revenue, registrations, and audience distribution stay visible from one admin workspace.

Website Information

What EventHub Verify is trying to improve

Instead of treating events as static cards only, EventHub Verify connects discovery, detail pages, registration, admin analytics, and account management into one consistent experience. The goal is to make the product feel closer to a real event platform than a one-screen demo.

Audience Students, attendees, and event admins
Project SE113.Q21 event registration website
Strength Readable event data, management flow, and analytics in one place
